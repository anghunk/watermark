## 屏幕水印工具

一个简单的小工具，可以在你的电脑屏幕上添加文字水印，非常适合录屏或在线分享时保护内容。

<img width="2559" height="1515" alt="image" src="https://github.com/user-attachments/assets/9f169ddf-570a-4732-8e02-855699cc3e92" />

## 主要功能

*   **自定义文字、大小、颜色、透明度和角度**。
*   **实时预览**：调整设置时，水印会立即变化。
*   **全屏或自定义区域**：水印可以覆盖整个屏幕，也可以只放在你指定的某个小区域。
*   **点击穿透**：水印不会影响你点击屏幕上的其他东西。

## 如何使用

**1. 直接运行**

在 [releases](https://github.com/anghunk/watermark/releases) 中下载 exe 文件，双击运行。

**2. 命令行执行**

在命令行中运行 `python watermark_gui.py`。
